
# C# Learning Repository
## Featured Project: BuddyBee Console Bot

BuddyBee is a lightweight non-API console chatbot built fully in C#.

This project was created to practice:
- C# fundamentals
- Methods and loops
- File handling
- Lists and history systems
- Input validation using TryParse
- Console-based UI interactions

## Features

- Calculator system
- String comparison
- Random programming jokes
- Real-time clock command
- Command history tracking
- Typing animation effect
- Colored console chat interface
- Username memory using file handling
- Safe integer validation
- Command aliases (calc/calculator)
- Calculator escape system using back command

## Technologies Used

- C#
- .NET Console Application
- File Handling
- Lists & Loops
- TryParse Validation

## Future Plans

- API Integration
- .NET Upgrade
- GUI/Web Version

## How to Run

1. Open the project in Visual Studio or VS Code
2. Compile and run the C# file
3. Start chatting with BuddyBee in the console
